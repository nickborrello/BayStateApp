# Track Spec: Build the Customer Storefront

## Goal
Create a high-performance, mobile-first customer-facing storefront for Bay State Pet & Garden Supply. This track focuses on the "Chewy meets local hardware store" experience with modern bento-grid layouts, intelligent search, and seamless product discovery.

## Requirements
- **Bento-Grid Homepage:** Modern, clean layout showcasing categories, featured products, and seasonal campaigns.
- **Intelligent Command Bar:** Fuzzy search optimized for 300+ brand catalog, handling typos and quick navigation.
- **Product/Category Pages:** Browse products by category with filters and sorting.
- **Product Detail Pages:** High-quality imagery, pricing, stock status, and "Add to Cart" functionality.
- **Service Integration:** Services displayed contextually alongside related products.
- **Mobile-First Design:** 44px+ touch targets, sticky cart, optimized for "barn-ready" use.

## Technical Constraints
- Next.js (App Router) with Server Components where possible
- TypeScript (Strict Mode)
- Supabase for data fetching
- shadcn/ui + Tailwind CSS
- Zustand for cart state management
- PWA-ready structure
